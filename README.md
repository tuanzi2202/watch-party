\# Watch Party 视频同步工具



这是一个支持多端的视频同步共享工具，允许朋友们在线上同步观看视频。



\## 项目结构



\- `/watch-party-server`: Node.js 后端服务，负责处理 WebSocket 连接和同步逻辑。

\- `/watch-party-client-pc`: 供 PC 端浏览器使用的网页前端。

\- `/watch-party-client`: 基于 React Native / Expo 开发的移动端应用。



\## 快速启动



进入各自的目录，安装依赖并启动服务：



```bash

cd watch-party-server

npm install

npm start

```
